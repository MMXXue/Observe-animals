The showSlides(slideIndex) function is used to display slides, and the parameter slideIndex is passed in when calling to display the first slide.

The plusSlides(n) function controls the display of the next or previous slide. Realize by calling showSlides(slideIndex += n), the n parameter can be positive or negative to control the display of the next or previous slide.

The currentSlide(n) function controls the slide show at a specific position. Realize by calling showSlides(slideIndex = n), set slideIndex to n, and display the slide at the corresponding position.

The showSlides(n) function is used to display slides. When the parameter n passed in is greater than the number of slides, slideIndex is set to 1, and when it is less than 1, slideIndex is set to the number of slides. Hide all slides by iterating through the array of slide elements, setting their display property to none. At the same time, traverse the array of slide dot elements, and replace "active" in their className attribute with an empty string. Then, set the display property of the current slide to block to display the current slide. Add "active" to the end of the "demo" property of the current slide, and highlight the "demo" of the current slide. Finally, set the innerHTML attribute of the slide description text to the alt attribute value of the current slide dot.


  • 2. Popup functionality for feedback.html

When the submit button is clicked, a prompt box will pop up on the page. The code first gets the form element whose ID is 'form_id' and stores it in a constant variable named form. Then, add a 'submit' event listener to the form. When the form is submitted, pass event.preventDefault() to prevent the default form submission behavior. Finally, call the alert() function to display a prompt box with the text 'Thanks for your support!' on the page.


  • 3. Load the Facebook SDK script for fish.html , cougar.html , snow_leopard.html , elephant.html , fox.html , littlepanda.html , alcedo.html , bear.html.

This code is used to load the Facebook SDK script to embed Facebook's social plug-ins in the webpage, such as like, share, comment and other functions.

Specifically, this code defines an anonymous self-executing function that accepts three parameters: d, s, id. where d represents the document object, s represents the script type to add to the document, and id represents the ID of the script element. The next code is to determine whether a script element with the same ID already exists in the document, and if it exists, it will not be loaded repeatedly.

If it doesn't exist, create a new script element via the document.createElement() method, add it to the document, and set its src attribute to the URL of the Facebook SDK. In this URL, some parameters are included. For example, #xfbml=1 means to enable the function of social plug-ins, and version=v3.0 means to use the SDK version.

Finally, the SDK is loaded by inserting the newly created script element before the first script element in the document via the parentNode.insertBefore() method.


  • 4. Carousel function for animal.html

The first part defines three variables: slides saves all the slides in the carousel, dots saves the dots below the carousel, currentSlide saves the subscript of the slide currently being displayed, and the initial value is 0.

The second part adds click events for all the dots. When the user clicks on a dot, the value of the currentSlide variable will be updated, and the setActiveSlide() method will be called to set the currently displayed slide and the currently selected slide according to the value of currentSlide. dots.

The third part realizes the function of automatically playing the slide show. An autoPlay() function is defined, which is called every 5 seconds. This function will add 1 to the value of currentSlide, and then call the setActiveSlide() method to set the currently displayed slide and the currently selected dot according to the value of currentSlide. When the value of currentSlide is equal to the length of slides, reset it to 0 and start playing from the beginning.


  • 5. Button click function for urban_animals.html

This code adds a click event listener to each button element with a class of 'btn' on the page. When a button is clicked, it gets the value of its 'data-href' attribute using the getAttribute method and stores it in a href variable. Then it sets the current window's location to the value of href, which redirects the user to the URL specified in the data-href attribute of the clicked button.

  • Normal or reading mode button

Notes: This button is placed at the top-right corner of a specific page within the website. Clicking it toggles between normal mode and reading mode. In reading mode, you cannot navigate to other pages of the website since these components are hidden. Moreover, the page in reading mode is also suitable for printing.
